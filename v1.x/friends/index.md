# Friends

A namespace to get information about users friends on Game Jolt

## URL Endpoint

```
/friends/
```

## Requests

| Name                             | Description             |
| -------------------------------- | ----------------------- |
| [**Friends**](/friends/fetch.md) | Lists a user's friends. |

## Example Uses

* List and count friends.
* Find currently online friends.
