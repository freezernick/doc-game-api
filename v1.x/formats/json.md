# Formats - JSON

Outputs data as a JSON string.

## Example

```
{
  "response":
  {
    "success":"true",
    "trophies":
    [
      {
        "id":"3",
        "title":"Cookie Magnet",
        "description":"Eat 10 points worth of cookies."
      },
      {
        "id":"4",
        "title":"I like this job!",
        "description":"Eat 20 points worth of cookies."
      }
    ]
  }
}
```
